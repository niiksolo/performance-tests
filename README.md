# 🚀 Performance Tests  

Этот проект запускает нагрузочные тесты с помощью **JMeter** и **k6**,  
а результаты автоматически публикуются в **GitHub Pages**.  

## 🛠 Технологии  
- ⚡ Apache JMeter  
- 📈 k6  
- 🤖 GitHub Actions (CI/CD)  
- 🌐 GitHub Pages (отчёты)
